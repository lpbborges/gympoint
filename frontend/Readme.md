## Front-end

<strong>Como instalar</strong>

Use git clone [git](https://github.com/lpbborges/gympoint-frontend/) para clonar o Gympoint.

```bash
git clone https://github.com/lpbborges/gympoint-frontend.git
```

<strong>Como executar</strong>

```bash
yarn start
```
